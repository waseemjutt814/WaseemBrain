# Roman Urdu Technical Dictionary

**Variable**: Variable programming main aik container hota he jo data store karta he. Jese `x = 10`, isme `x` aik variable he jisme humne 10 store kiya he.

**Function**: Function code ka aik block hota he jo specific kaam karta he. Jab bhi us kaam ki zarorat ho, hum pura code likhne ki bajaye sirf function ko "call" (bula) lete hain. Ye code reusability k liye bohat faida mand he.

**Loop (For/While)**: Loop ka matlab he kisi code ya kaam ko bar bar dohrana jab tak koi specific shart (condition) puri na ho jaye. `for` loop tab use hota he jab humein pehlay se pata ho k kitni bar chalana he, aur `while` loop tab jab humein condition match hone ka intezar ho.

**Class & Object (OOP)**: Class aik blueprint ya naqsha hota he. Maslan, "Car" ki aik class he. Object us naqshe se bani hui asal gaari hoti he jese "Honda Civic". Object-oriented programming main hum har cheez ko objects samajh kar banate hain.

**API (Application Programming Interface)**: API do softwares ko aapas main baat karne ka rasata deti he. Jese weather app API se current weather ka data mangwati he.

**Algorithm**: Algorithm kisi masle (problem) ko solve karne ka step-by-step formula ya tareeqa hota he.

**Bug**: Code main koi ghalti (error) ya aisi cheez jo expected tor pe kaam na kar rahi ho, usko bug kehte hain. Isay thek karne k process ko "Debugging" kehte hain.

**Database**: Database wo jagah he jahan website ya app ka sara information store hota he jese users k passwords aur messages. MySQL aur SQLite databases ki misalein hain.
